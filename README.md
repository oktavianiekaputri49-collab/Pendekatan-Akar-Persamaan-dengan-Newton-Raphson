# Pendekatan-Akar-Persamaan-dengan-Newton-Raphson
Mini project metode numerik menggunakan Metode Newton-Raphson untuk melakukan pendekatan akar persamaan nonlinear f(x)=x^3 −x−2.  Dalam project ini dilakukan proses iterasi numerik menggunakan turunan fungsi untuk memperoleh akar persamaan secara cepat dan akurat dengan bantuan Microsoft Excel.
